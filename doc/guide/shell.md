# Shell 使用手册

本文面向使用者，说明如何在 WebLens 中进入 Pod Shell 并进行基本排障。

## 打开 Shell

1. 进入 Pods 列表
2. 在目标 Pod 行点击三点菜单
3. 选择目标容器后打开 Shell
4. 底部工作区会新增一个 Shell 标签

## Shell 基本能力

- 支持多 Pod / 多容器并行标签
- 支持右键复制/粘贴
- 支持 Ctrl+V 原生粘贴
- 长命令到达右边界后自动换行

## 重连按钮

Shell 标题栏提供“重连”按钮：

- 无论当前状态是“连接中”还是“已连接”，点击都会立即打断并重建连接
- 不会清空历史输出
- 新会话会在历史输出下方继续显示新的 prompt

## 常见问题

### 显示连接错误/连接关闭

常见原因：

- Pod/容器瞬时重启
- 网络抖动或代理层中断
- 权限（pods/exec）限制

建议先点击“重连”重试，再检查后端日志与容器状态。
