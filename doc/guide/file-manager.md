# 文件管理面板使用手册

文件管理面板位于 Shell 标签右侧，用于对当前 Pod/容器文件系统进行基础操作。

## 打开与收起

- 默认收起为窄侧边栏（箭头按钮）
- 点击展开后显示“文件管理”窗口
- 可再次收起，不影响 Shell 继续使用

## 路径导航

支持两种方式：

- 面包屑点击跳转（分隔符为 `<`，目录名为蓝色下划线）
- 手动输入路径并点击“进入”

当输入路径不存在时，会显示红色提示：`路径不存在，请检查`

## 工具栏能力

- 上一级
- 刷新
- 上传
- 下载（多选）
- 删除（多选）
- 重命名（单选）
- 新建文件夹

按钮状态说明：

- 未勾选时，下载/删除/重命名会置灰不可点击
- 重命名仅在单选时可用

## 列表说明

- 支持多选框
- 支持目录/文件类型区分
- 点击目录名称可进入子目录
- 文件默认不直接打开

## 限制说明（第一版）

- 不支持在线编辑文件
- 不支持拖拽上传与断点续传
- 下载依赖容器内 `tar` 命令
- 极简镜像可能缺少部分命令，出现失败时会显示错误提示
