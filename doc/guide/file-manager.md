# 文件管理面板使用手册

文件管理面板位于 Shell 标签右侧，用于对当前 Pod/容器文件系统进行基础操作。

## 打开与收起

- 默认收起为窄侧边栏（箭头按钮）
- 点击展开后显示“文件管理”窗口
- 可再次收起，不影响 Shell 继续使用

## 路径导航

顶部为**单条地址栏**（不再分「面包屑一行 + 输入框一行」）。

**默认：面包屑**

- 显示当前路径层级，例如：`/` `›` `appdata` `›` `appuser`
- 点击某一节即可跳到该层目录
- 路径很深时，地址栏中间区域可横向滑动查看

**手动输入路径**

- 点击面包屑右侧的**空白区域**，或**双击**地址栏（不要点在某一节目录名上）
- 整条地址栏会变成输入框，并自动选中全部路径文本，便于直接改
- 按 **Enter** 跳转到输入路径
- 按 **Esc** 或点击别处导致**失焦**：取消编辑，恢复面包屑（未按 Enter 的修改不会生效）

当输入路径不存在时，会显示红色提示：`路径不存在，请检查`

## 工具栏能力

- 上一级
- 刷新
- 上传
- 下载（多选）
- 删除（多选）
- 重命名（单选）
- 新建文件夹

按钮状态说明：

- 未勾选时，下载/删除/重命名会置灰不可点击
- 重命名仅在单选时可用

## 列表说明

- 支持多选框
- 支持目录/文件类型区分
- 点击目录名称可进入子目录
- 文件默认不直接打开

## 传输任务

工具栏下方为 **传输任务** 区域（可折叠）：

- **上传**：支持多选文件；展示文件名、进度条与百分比（axios 上传进度）；失败时显示错误摘要
- **下载**：流式打包为 tar；若响应含 `Content-Length` 则为**真实**进度；否则在列表能汇总勾选项**已知大小**时显示**估算**进度（文案会标明「估算」）；无法估算时仅显示已接收字节，不伪造百分比

任务较多时列表区域内可纵向滚动；完成后会保留一段时间并自动收起，也可手动关闭单项。

## 限制说明（第一版）

- 不支持在线编辑文件
- 不支持拖拽上传与断点续传
- 下载依赖容器内 `tar` 命令
- 极简镜像可能缺少部分命令，出现失败时会显示错误提示
